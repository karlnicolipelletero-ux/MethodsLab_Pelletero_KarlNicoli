Description
This program demonstrates the use of Java Methods to perform various tasks. It covers declaring methods, passing parameters, returning values, and using loops.


Included Files
MethodsLab.java: 
The main source code containing all 5 required tasks.

Features
The program implements the following 5 tasks:

1. Greetings: A simple void method that prints a welcome message.

2. Calculator: Methods for Addition, Subtraction, Multiplication, and Division.

3. Factorial: Calculates the factorial of a user-input number.

4. Array Sum: Calculates the sum of an integer array.

5. Prime Checker: Determines if a number is prime or not.
